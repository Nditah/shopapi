# shopapi

### Status
[![Build Status](https://travis-ci.org/Nditah/shopapi.svg?branch=master)](https://travis-ci.org/Nditah/shopapi)

1. Clone the Repo

> git clone https://github.com/Nditah/shopapi.git

2. Install Dependencies

> npm install

3. Start the App

> npm start

4. Test the App

> npm test

5. Build and Serve App

> npm serve
